"You will have to build a CRUD application in a Python web framework, that allows a
user to login using the Oauth protocol, with GitHub as the provider.

Once logged in, they must be able to create/edit/delete/update their personal
profile, e.g, name, past and previous address, phone number etc.

    Please make sure the repository for the project is publicly hosted on GitHub
    Please make it easy to build, test and run your application

The project review will focus on the following points:

    Clarity and intent within both production code, and test suites
    Predictability of system behaviour
    The amount of code written (less is more)
    Coding standards/best practices
    Ease of setup/running - documentation"

Create a CRUD application (Github app)

    - Python web framework (Flask)
    - Create a Github app
        https://developer.github.com/apps/quickstart-guides/setting-up-your-development-environment/
        https://github.com/swinton/github-app-demo.py/blob/master/github/apps/installation.py (Python example)


    - Login using Oauth protocol for GitHub
    - Once logged in, it must be able to do these actions in a profile:
        - create
        - edit
        - delete
        - update
            - name
            - past and previous address
            - phone number

    - Test application using unittest
    - The repository must be public


Create a virtual environment
    python3 -m venv venv (within the project root folder)
    source venv/bin/activate (activate environment)
    which python (check venv. E.g. /home/sisco/Workspace/Python/wolf&badget-github/venv/bin/python)

Running Flask application
    export FLASK_APP=hello.py
    python -m flask run
    Running on http://127.0.0.1:5000/
